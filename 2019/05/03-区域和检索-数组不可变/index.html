LeetCode-303-区域和检索-数组不可变

2019-05-03 LeetCode Comments

给定一个整数数组nums，求出数组从索引i 到 j (i ≤ j) 范围内元素的总和，包含i, j 两点。

例子

说明

  1. 你可以假设数组不可变。
  2. 会多次调用 sumRange 方法。

解法

1. 直接循环

遇到这种题目脑子里第一个想到的自然就是直接循环从i累加到j，但是题目中说明了

会多次调用 sumRange 方法

因此每次调用该方法时都需要重新累加一遍，十分耗时。

代码

2. 提前计算好和

可以提前构建一个sums[]数组，数组里的第i个元素是从0到i个元素的和，这样，计算第i到j的和只需要求sums[j]-sums[i-1]的值，耗费常数时间。当进行多次调用时较为节约时间。
